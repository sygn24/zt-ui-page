# 禁用状态

按钮不可用状态。

默认按钮 虚线按钮 主要按钮 信息按钮 成功按钮 警告按钮 危险按钮

</div>
<script>
    export default {
        methods: {
            handleClick() {
                this.$msg('点击了')

        }
    }
</script>

# 图标按钮

带图标的按钮可增强辨识度（有文字）或节省空间（无文字）。

设置 icon 属性即可，icon 的列表可以参考 icon 组件，也可以设置在文字右边的 icon ，只要使用zt-icon标签即可，可以使用自定义图标。

# 加载状态

点击按钮后进行数据加载操作，在按钮上显示加载状态。

信息按钮 成功按钮 警告按钮 危险按钮

要设置为 loading 状态，只要设置loading属性为true即可。

<div class="row">
    <zt-button type="info" :loading="loading" @click="loadingClick">信息按钮</zt-button>
    <zt-button type="success" loading>成功按钮</zt-button>
    <zt-button type="warning" loading>警告按钮</zt-button>
    <zt-button type="danger" loading>危险按钮</zt-button>
    <zt-button circle loading></zt-button>
</div>
<script>
    export default {
        data() {
            return {
                loading: false
            }
        },
        methods: {
            loadingClick() {
                this.loading = true
                setTimeout(() => {
                    this.loading = false
                }, 2000)
            }
        }
    }
</script>
